What Happens

  1. Build a seed from the controls below
  2. Run `tools/randomize.js` or `tools/hack_workdir.js` on the server
  3. Patch data, build `out.bin`, and apply generated minimap tile, map, and marker-path patches when Tracks is enabled
  4. Watch progress in the Live Console

Best Default

On master, the safe/default randomized ROM path is workspace-based so the root source tree stays canonical. Use Workspace Build or the default tools/randomize.js flow for playable randomized ROMs.

Suggested Flow

1

Pick A Preset

Start in the Randomized ROM panel and choose Tracks Only, Full Safe Build, Workspace Build, or Championship Focus.

2

Run It

Click Generate Randomized ROM. Preview mode only updates data files and validation output; a real build is required for the generated minimap art and marker path to appear in the ROM.

3

Watch Output

Follow progress in the Live Console to see validation, data writes, and build status as they happen.

4

Verify Or Restore

Use the built-in verify and restore actions once the randomizer completes.
